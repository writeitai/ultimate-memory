  During the analysis steps, you can also use Codex and Antigravity alongside your subagents.
  - when the agent should use CODEX, it means you will execute the agent as `codex exec --yolo --model gpt-5.5 PROMPT="<your prompt>"`
  - when the agent should use ANTIGRAVITY, it means you will execute the agent as `agy --dangerously-skip-permissions -p "<your prompt>"`

  All analyses should be written down to markdown files (feel free to create a dedicated directory).
