1"""The HTTP API surface (WP-1.6): the query engine's primitives over FastAPI. 

2 

3A thin, typed veneer: every endpoint delegates to one QueryEngine primitive 

4and returns the D49 envelope verbatim. The app is built per composed engine 

5(profiles own composition); the surface itself never touches adapters. 

6"""

17def build_api(*, engine: QueryEngine, deployment_id: UUID) -> FastAPI:
